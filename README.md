# 6426-ConwaysGameOfLife
Instructions:
Set "Grid" as startup project
Run application

Interactions:
Mouse click, click-drag: Set cells to Alive.
Escape: Quit.
Space: Start/Pause evolutions.
Tab: Fill randomly.
Enter: Clear.
